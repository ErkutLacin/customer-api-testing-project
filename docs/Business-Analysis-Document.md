# 1. Project Overview

This project has been prepared as a portfolio study to demonstrate the end-to-end analysis, documentation and API testing processes performed by a Business Analyst.

The project simulates the analysis and testing activities of a Customer Management API in a digital banking environment. Throughout the project, business requirements, functional requirements, business rules, API requirements, test scenarios and API tests will be created step by step.

The main objective is not to develop a backend application, but to demonstrate a Business Analyst's analytical thinking, documentation skills and testing approach through a realistic case study.
