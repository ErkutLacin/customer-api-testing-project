## Swagger Dokümantasyonu

Bu proje kapsamında hazırlanan API dokümantasyonu, **OpenAPI 3.0** standardına uygun olarak geliştirilmiştir.

Swagger dokümantasyonu içerisinde;

- Endpoint açıklamaları
- Request ve Response modelleri
- Parametre açıklamaları
- Örnek istekler (Request Examples)
- Örnek yanıtlar (Response Examples)
- HTTP durum kodları
- Hata senaryoları

yer almaktadır.

### Swagger'ı Görüntüleme

1. [`api/openapi.yaml`](api/openapi.yaml) dosyasını açın.
2. Dosya içeriğini kopyalayın.
3. https://editor.swagger.io adresini açın.
4. Varsayılan örneği silerek kendi `openapi.yaml` dosyanızı yapıştırın.

Swagger Editor, API dokümantasyonunu otomatik olarak oluşturacaktır.
